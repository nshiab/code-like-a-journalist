# Installation

Commençons par installer tout ce dont nous avons besoin.

## Un éditeur de code

Le code n'est que du texte. Ça n'a rien de magique. En théorie, vous pourriez écrire votre code dans n'importe quel éditeur de texte, mais il est plus pratique d'utiliser un éditeur de code qui offre de nombreuses fonctionnalités très utiles.

Il y a de nombreuses options gratuites et intéressantes, mais le plus populaire est Visual Studio Code, et c'est celui que nous allons utiliser. C'est un projet maintenu par Microsoft.

Téléchargez-le depuis le [site web](https://code.visualstudio.com/) et installez-le.

Une fois installé, ouvrez-le. Ça devrait ressembler à ceci :

## Un environnement d'exécution JavaScript

Pour que votre ordinateur comprenne et exécute du JavaScript/TypeScript, vous avez besoin d'un environnement d'exécution JavaScript.

Encore une fois, vous avez de nombreuses excellentes options en code ouvert, mais je trouve Deno particulièrement facile à utiliser. Deno est maintenu par Deno Land Inc., avec Ryan Dahl comme PDG. Dahl a également créé Node.js, un autre environnement d'exécution en code ouvert très populaire.

Pour l'installer, visitez https://deno.com/ et copiez la commande appropriée pour votre système d'exploitation.

Ne vous inquiétez pas si vous avez une version différente. Tant que c'est une v2.x.x, tout ira bien.
